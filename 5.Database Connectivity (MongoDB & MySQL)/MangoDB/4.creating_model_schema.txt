## **âœ… 1ï¸âƒ£ What is a Mongoose Schema?**

ðŸ‘‰ **A schema is like a blueprint** for how data should be stored in MongoDB.  
ðŸ‘‰ It helps **define fields, data types, validation rules, and relationships** between documents.  

âœ… **Example Schema for a User Collection:**

const UserSchema = new mongoose.Schema({
  name: { type: String, required: true }, // Name is required
  email: { type: String, unique: true, required: true }, // Email must be unique
  age: { type: Number, required: true, min: 18 }, // Age must be 18 or above

});

```

}
```
